# Installation

## Stable release

To install sinly_quant, run this command in your terminal:

```sh
uv add sinly_quant
```

Or if you prefer to use `pip`:

```sh
pip install sinly_quant
```

## From source

The source files for sinly_quant can be downloaded from the [Github repo](https://github.com/stevenchen521/sinly_quant).

You can either clone the public repository:

```sh
git clone git://github.com/stevenchen521/sinly_quant
```

Or download the [tarball](https://github.com/stevenchen521/sinly_quant/tarball/master):

```sh
curl -OJL https://github.com/stevenchen521/sinly_quant/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd sinly_quant
uv pip install .
```
